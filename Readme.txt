# Matrimonial Fake Profile Detection System

## 📌 Project Overview

The Matrimonial Fake Profile Detection System is an AI-powered web application designed to identify and detect fake matrimonial profiles using Machine Learning techniques. The system analyzes user profile information and predicts whether a profile is genuine or suspicious.

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/Reddy1727/matrimonial-fake-profile-detection.git
```

### Frontend Setup

```bash
cd frontend
npm install
npm start
```

### Backend Setup

```bash
cd backend
pip install -r requirements.txt
python app.py
```

### Machine Learning Setup

```bash
cd ml-model
pip install -r requirements.txt
python train.py
```

---

## 🤖 Machine Learning Module

The ML module analyzes profile attributes such as:

* Age
* Education
* Occupation
* Income
* Profile Completeness
* Activity Patterns

and predicts whether a profile is:

* Genuine Profile
* Fake Profile
